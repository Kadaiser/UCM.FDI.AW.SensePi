Services

Mediante nuestra estación de recopilacion de datos EMMA (Estacion Movil de Medicion Medioambiental) compuesta de distintos sensores de medición obtendremos datos que serán almacenados en una base de datos. Esta dará servicio a la aplicación web que administrará, para que mediante un entorno amigable y accesible, los usuarios puedan consultarlos. Ofreceremos mapas, estadísticas, gráficos, tablas, etc. Que tendrán distintos grados de visibilidad según el usuario.
